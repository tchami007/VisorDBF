# Milestone v1.0: Visor y Exportador DBF

**Status:** ✅ SHIPPED 2026-07-10
**Phases:** 1-4
**Total Plans:** 17

## Overview

Aplicacion de escritorio Windows (C# / .NET 8 / WPF) que permite abrir archivos DBF (dBASE III, IV, FoxPro), visualizar su contenido en una grilla tabular y exportarlo a TXT con separadores, formatos por columna y codificaciones configurables.

## Phases

**Goal:** Tener una solucion C# / WPF compilable con la arquitectura base (Core + UI), capacidad de abrir y leer archivos DBF correctamente — incluyendo deteccion de codificacion via Language Driver ID — y visualizar su contenido en la grilla principal.

- [x] 1.3 — Servicios de lectura DBF (IDbfReaderService, DbfReaderService con DbfDataReader, EncodingDetectionService) ✓ 2026-07-06

**Goal:** El usuario puede configurar todos los parametros de exportacion (separadores, encabezado, filas, codificacion de salida) y exportar el contenido completo a un archivo TXT de forma asincrona con indicador de progreso y capacidad de cancelacion.

- [x] 2.3 — Dialogo de progreso de exportacion (ExportProgressDialog, barra determinista, boton cancelar) ✓ 2026-07-07
- [x] 2.4 — Integracion: flujo completo apertura → configuracion → exportacion → confirmacion ✓ 2026-07-07

**Details:**

## Milestone Summary

**Key Decisions:**
- .NET 8 LTS como target framework — Validated
- WPF con MVVM — Validated
- DbfDataReader 0.4.3 para lectura DBF — Validated
- System.Text.Json para configuracion — Validated
- Publicacion self-contained — Validated
- Separacion Core/UI — Validated
- Language Driver ID como codificacion por defecto — Validated
- CurrentCulture para grilla, InvariantCulture para exportacion — Validated

**Issues Resolved:**
- EncodingPickerDialog SelectedItem/SelectedValue type mismatch
- StatusMessage incorrecto al cancelar con archivo previo
- LoadPreviewAsync no llamado en constructor
- Progress<T> creado en thread pool sin SynchronizationContext
- Ventana en blanco por falta de binding en Visibility
- MaxLength=1 en separador eliminado para multi-caracter
- SqlExportService eliminado, funcionalidad consolidada en SybaseExportService

**Technical Debt Incurred:**
- Logs de FileLogger en %TEMP% sin limpieza automatica (aceptado)
- Password de Sybase no persiste entre sesiones (seguridad por diseno)
